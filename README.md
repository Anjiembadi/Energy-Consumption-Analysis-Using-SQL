--  Energy Consumption Analysis (SQL Project)
-- Objective:
--   To analyze global energy usage, GDP trends, and CO2 emission data 
--   using structured SQL queries for meaningful insights.

-- Key Highlights:
--   • Designed a star schema database with six interlinked tables 
--     (Country, Energy, CO2, GDP, Year, and Sector).
--   • Imported and cleaned large datasets for multiple countries 
--     to ensure referential integrity.
--   • Used complex SQL joins, subqueries, and aggregate functions 
--     to calculate country-wise and year-wise energy efficiency ratios.
--   • Analyzed correlation between GDP growth and CO2 emissions 
--     to identify top and low-performing economies.
--   • Created views and temporary tables for reusable analytical queries.
--   • Derived insights showing high-GDP nations consume and emit more energy, 
--     recommending renewable adoption to balance growth and sustainability.

-- Tech Stack:
--   MySQL | Data Modeling | Star Schema | Joins | Group By | Subqueries | Aggregations
